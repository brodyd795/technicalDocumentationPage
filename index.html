Regular Expressions Documentation

Introduction

When dealing with written language, we often need an easy way to search through the text to find or replace particular strings. For example, you might find yourself needing to find the first or all instances of a data (e.g., 2019-06-10), or you might need to change such strings to a different format (e.g., 6/10/19). Regular expressions were developed for exactly this purpose.

This document will walk you through the basics of how regular expressions ('regex' for short) are formed, focusing on the replacement use. Once you've mastered the basics, you can apply this knowledge to the programming language of your choice (e.g., Python, JavaScript, etc.) to work with texts and create useful applications.

For this documentation, regular exprssions will be presented in the /matchString/replaceString/ format, where the first string is what you are looking to match, and the second is what you'd like to replace each match with.

Getting Started

There are lots of websites available for testing out your regular expressions, such as Regex101.com. On that site, there are fields for the match string and the substitution string, as well as your own test string. There's also a handy Quick Reference guide in the lower right corner, most of which we will walk through below.
